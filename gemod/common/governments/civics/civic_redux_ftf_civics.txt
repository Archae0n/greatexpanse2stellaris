#CIVICS FOR FORGING THE FUTURE

#MARCH OF SCIENCE - 10% Research Point Gains. UNIQUE EDICT: March of Science - Applies a bonus based on Fanatic Ethic | UNIQUE TECH: Progress Day Parades - Applies a permanent version of March Of Science's buff.
#PROGRESS SHARED - 25% Subject Technology Sharing Bonus. Gain the Epigenetic Triggers tech if you start the game with this Civic.
#FOCUS OF KNOWLEDGE - At the start of the Game, choose a Tech Category. You gain 25% Bonus Research Speed to all Techs in that category. Other Techs lose 10% Research Speed. 
#DIGITAL LIFE - 10% Happiness, 10% Governing Ethics Attraction and 20% Bonus Unity. +10% Building Maintenance and -5% Energy Gain. 
#TEST SUBJECTS - 5% Faster Research. Gain the 'For Science' Planet Edict - Removes a Xeno Pop, and applies a 20% Research Speed buff for 5 Years. Purged Pops provide Research Points. 
#SELF-ENHANCING WAREHOUSES - Can build Self-Enhancing Warehouses, which provide Mineral/Energy Storage, and if a Robot is nearby, a bonus to all Mineral/Energy production on the Planet. 
#AWAKENED MINDS (UTOPIA) - +1 Tech Alternative, and double the efficiency of the Technological Ascendancy Perk. Gain an additional bonus based on the first Ascension Path Perk you take. 
#AGE OF STEEL (UTOPIA/SYNTHETIC DAWN) - -20% Robot Build/Maintenance Cost and Build Robots 20% Faster. This bonus increases as you go down the Synthetic Ascension Path. 
#ROBOTICISTS - +1 Robot Trait Point and 15% Cheaper/Faster Robots. Robot Cost and Build Speed modifier is doubled if the Machinists Civic is enabled.

#LIVING CORE (HIVE MIND/LEVIATHANS) - Can build Living Amplifiers on Planets to give 5% Army Health, Damage and Morale. The Hive's Consiousness manifests as a powerful Defense Army. If you lose control of your HomeWorld, kill all Pops in your empire. 
#UNITED KNOWLEDGE (HIVE MIND) - At the start of the Game, or shortly after selecting it, choose a Tech Category. You gain 15% Research Speed to all Techs in that Area. After 15 Years, it falls into a persistent 5% Bonus and choose a new Tech Category. 

#THE CORE (MACHINES) - Begin the game on a 25-Tile Machine World with enhanced resources

civic_march_of_progress = {
	modification = yes #Can you add it to/remove it from your empire after the game starts?
    
	playable = { #What do you need to make this Civic useable in the game? (Usually reserved for DLC - for example "Utopia" makes it exclusive to those with the Utopia DLC)
    }
    
	potential = { #What do you need to see it? (Mainly used to seperate Hive Mind civics from non-HM)
        ethics = {
			NOT = {
				value = ethic_gestalt_consciousness
			}
		}
    }
	
	possible = { #What conditions do you need to use it on your empire?
		ethics = {
			OR = {
				text = civic_tooltip_any_fanatic
				value = ethic_fanatic_militarist
				value = ethic_fanatic_xenophobe
				value = ethic_fanatic_egalitarian
				value = ethic_fanatic_materialist
				value = ethic_fanatic_pacifist
				value = ethic_fanatic_xenophile
				value = ethic_fanatic_authoritarian
				value = ethic_fanatic_spiritualist
			}
		}
	}
	
	random_weight = { #What are the odds the AI will pick it?
		base = 1
	}

	description = "civic_tooltip_march_of_science_effects" #Any custom effects?
	modifier = { #What does this civic do?
		country_resource_physics_research_mult = 0.1
		country_resource_society_research_mult = 0.1
		country_resource_engineering_research_mult = 0.1
	}
}
